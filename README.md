# A+ Assistant

A+ Assistant 是一款 Windows 桌面股票与基金行情助手，基于 Tauri 2、Vue 3、TypeScript 和 Rust 构建。应用提供股票和场外开放式基金自选、实时刷新、详情图表、本地持仓账本、账户收益汇总、拖拽排序、窗口置顶、自启动和托盘常驻等功能。

> 当前正式版本：`1.0.3`
>
> 发布日期：2026-07-23
>
> 支持系统：Windows 10 / Windows 11 x64

## 正式安装包

- 安装格式：NSIS x64
- 安装包：`src-tauri/target/release/bundle/nsis/A+助手_1.0.3_x64-setup.exe`
- 默认目录：存在 D 盘时优先使用 `D:\A+助手`，否则使用当前用户的本地应用目录
- 升级安装：检测到旧版本时显示中文升级提示
- 用户数据：升级时默认保留自选、排序、设置、持仓和交易记录
- 图标：安装包、程序、窗口、任务栏和托盘使用项目内统一图标资源

- 新增基金完整详情页，支持关联板块、业绩走势、我的收益和本地持仓摘要。
- 使用完整历史净值绘制近 1 月、近 3 月、近 6 月、近 1 年和全部区间走势。
- 修复基金历史净值日期按 UTC 解析导致日期提前一天的问题。
- 基金历史净值缓存会根据最新确认净值日期自动失效，避免已确认数据未刷新。
- 业绩图以基金和沪深 300 的首个共同交易日为起点计算区间收益。
- 基金“近一年”优先读取官方基金基础信息；字段缺失时根据完整净值历史计算，避免显示 `--`。
- `014855` 已验证近一年收益显示为 `+179.19%`。
- 基金列表展示主题板块，不再使用确认时间占用涨跌幅下方的信息位。
- 行业配置、主题板块和重仓股采用独立缓存；单项接口失败不影响其他详情区域。
- 支持基金加仓、减仓、修改持仓、交易记录编辑和删除，并按已确认净值重建收益快照。

### 股票行情与详情

- 自动刷新遇到空响应或单条失败时保留旧行情，避免列表偶发显示“暂无数据”。
- 股票搜索按数据源实际编码解码，修复中文候选项乱码。
- 修正 K 线红绿涨跌方向、总市值与流通市值对应关系。
- 详情页新增量比和换手率；最高价使用红色、最低价使用绿色，量比大于等于 1 显示红色。
- 买卖五档按 A 股习惯展示价格颜色、买卖深度背景和中间委比线。
- 分时图使用真实均价线，并过滤 15:00 后的非交易时段数据。
- 分时成交量由累计量转换为每分钟成交量，使用红绿柱展示。
- 加高分时量区域，使成交量约占图表四分之一，同时保留完整时间轴和五档区域。

### 界面与可靠性

- 标题在 `A+Stock Assistant` 和 `A+Fund Assistant` 之间切换股票、基金模式。
- 股票与基金分别保存自选、排序和持仓数据。
- 卡片支持专用拖拽手柄排序，并保留右键置顶、上移、下移和移到底部。
- 删除按钮只在悬停、焦点或选中状态出现，默认状态下行情数据靠右显示。
- 持仓数据仅在鼠标悬停卡片时展开，移出后恢复行情摘要。
- 股票和基金存在有效持仓时，顶部显示账户资金、当日预计收益和账户总收益。
- 设置页的置顶、自启动和恢复默认增加执行中状态、失败回滚和启动重放。
- 请求层统一超时、状态检查和错误处理，降低外部接口异常导致白屏或空数据的风险。
- 增加模态框焦点管理、键盘关闭和基础无障碍标签。
- 保留 WebView2 CSP 与本地设置清洗，降低升级后白屏和异常主题配置风险。

## 使用说明

### 股票模式

- 点击标题切换到 `A+Stock Assistant`。
- 支持按股票名称或六位代码搜索 A 股。
- 支持添加、删除、手动刷新、自动刷新和拖拽排序。
- 列表显示名称、代码、涨跌幅、现价和分时缩略线。
- 点击股票卡片打开详情；详情支持分时、日 K、周 K、月 K、季 K 和年 K。
- 详情展示大盘指数、价格指标、成交指标、均线、分时量和买卖五档。
- 右键股票可录入成本价和股数，自动计算市值、成本、盈亏和收益率。
- 存在持仓时，悬停卡片显示持仓详情，盈亏按红绿颜色区分。

### 基金模式

- 点击标题切换到 `A+Fund Assistant`。
- 仅支持场外开放式基金，不把 ETF、LOF、场内基金或股票接口中的基金结果混入列表。
- 支持按基金名称或六位代码搜索、添加、删除、刷新和排序。
- 列表展示基金名称、代码、估算涨跌幅、估算净值和主题板块。
- 点击基金卡片打开详情，查看确认净值、完整历史业绩、沪深 300 对比、行业配置和重仓股。
- 右键基金可建立初始持仓；详情页支持加仓、减仓、修改持仓和维护交易记录。
- 收益按官方确认净值更新，盘中估算只用于当前展示，不写入历史收益快照。

### 排序与交互

- 从卡片左侧手柄开始拖动，移动超过 6px 后进入排序状态。
- 拖动期间会高亮目标位置，并禁用误触详情。
- 右键菜单保留置顶、上移、下移和移到底部，作为拖动排序的备选。
- 股票和基金排序相互独立并持久化保存。

### 设置与窗口

- 支持深色、浅色和自定义主题。
- 支持背景透明度、背景颜色、字体颜色和字体设置。
- 支持显示或隐藏大盘指数、股票分时缩略线。
- 支持窗口置顶、开机自启动、最小化到托盘和恢复默认。
- 置顶和自启动调用失败时会回滚界面状态。
- 恢复默认会先同步关闭置顶和自启动，再重置本地设置。
- 应用启动后会重新应用已保存的窗口置顶状态。

## 数据与本地存储

当前版本使用公开接口获取行情、搜索和基金披露数据：

- 股票行情：腾讯行情相关接口
- 股票分时与 K 线：腾讯 / IFZQ 相关接口
- 股票搜索：新浪搜索建议接口
- 基金搜索、资料、净值、行业和持仓：东方财富相关接口
- 基金盘中估算：天天基金估算接口
- 基准指数：沪深 300 日线数据

公开接口不由本项目维护，其字段、可用性、频率限制和更新时间可能变化。应用会尽量保留最近一次有效数据，并让单条资产或单个详情区域的失败保持隔离。

自选、排序、设置、持仓和交易记录保存在本机 WebView2 / localStorage 相关目录中，不会上传到本项目服务器。

基金盘中数据是估算净值和估算涨跌幅，不等同于交易所逐笔价格，也不作为已确认收益入账。

## 安装与升级

1. 运行 `A+助手_1.0.3_x64-setup.exe`。
2. 安装器存在 D 盘时默认选择 `D:\A+助手`。
3. 检测到已安装版本时，安装器会用中文提示升级。
4. 升级默认保留所有本地数据。
5. 卸载时只有主动选择删除数据，才会清理本地应用数据。

### 白屏排查

当前版本已允许 Tauri IPC 所需的 `ipc:` 和 `http://ipc.localhost`，并在启动时清洗异常设置。如果仍出现白屏：

1. 结束所有 `A+助手` 进程后重新启动。
2. 重新安装最新正式版。
3. 仍无法恢复时，备份必要数据后清理 `%LOCALAPPDATA%\com.stock.widget\EBWebView`。

### 行情显示 `--`

- 非交易时段、基金暂停估算或数据源暂时缺少字段时，部分盘中值会显示 `--`。
- 应用会保留已有有效行情，不会因为一次空响应清空整个自选列表。
- 基金确认净值、盘中估算和历史净值属于不同更新时间的数据，请以基金公司最终公告为准。

## 开发环境

要求：

- Node.js 18+
- Rust stable
- Windows 10 / Windows 11
- Microsoft Edge WebView2 Runtime

安装依赖：

```bash
npm install
```

启动前端：

```bash
npm run dev
```

启动桌面开发环境：

```bash
npm run tauri:dev
```

## 项目结构

```text

├─ .github/workflows/       # CI 检查
├─ src/
│  ├─ api/                  # Tauri 命令调用与类型

│  ├─ icons/                # 程序、窗口、任务栏、托盘和安装包图标

│  ├─ src/http.rs           # 统一 HTTP 请求与解码

│  └─ tauri.conf.json       # Tauri 与打包配置
├─ package.json
├─ vite.config.ts
└─ uno.config.ts
```
