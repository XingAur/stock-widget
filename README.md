# A+ Assistant

一个基于 `Tauri 2 + Vue 3 + TypeScript` 构建的 Windows 桌面股票小组件，面向 A 股场景，强调常驻桌面、低打扰和快速查看。

它适合作为一个轻量行情侧边工具使用：

- 常驻桌面查看自选股实时行情
- 一键打开股票详情页
- 查看分时图、K 线、五档买卖盘
- 在不离开当前工作流的情况下快速掌握市场变化

## 特性

- 自选股管理
  - 添加、删除、排序自选股票
  - 股票名称 / 代码搜索
- 实时行情
  - 最新价、涨跌额、涨跌幅、最高、最低、开盘、昨收
  - 30 秒自动刷新
- 详情页
  - 分时图
  - 日 K / 周 K / 月 K / 季 K / 年 K
  - MA5 / MA10 / MA20 / MA30
  - 五档买卖盘
  - K 线支持滚轮缩放与拖动平移
- 桌面小组件体验
  - 详情页默认在右侧展开
  - 当右侧空间不足时自动改为左侧展开
  - 关闭详情后小组件保持原侧位置
- 界面与窗口设置
  - 深色 / 浅色 / 自定义主题
  - 背景透明度调节
  - 字体切换
  - 大盘指数显示开关
  - 分时 sparkline 显示开关
  - 窗口置顶
  - 最小化到托盘
  - 开机自启

## 技术栈

## 运行环境

- Windows 10 / Windows 11
- Node.js 18+
- Rust stable

建议使用最新版 `npm` 与 `rustup`。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发环境

```bash
npm run tauri:dev
```

如果你只想调试前端界面，也可以先启动：

```bash
npm run dev
```

### 3. 构建桌面应用

```bash
npm run tauri:build
```

## 验证与维护

前端纯逻辑使用 Vitest 覆盖，主要包括格式化与图表聚合逻辑：

```bash
npm run test
```

前端类型检查与生产构建：

```bash
npm run build
```

也可以一次运行前端测试与构建：

```bash
npm run check
```

Rust / Tauri 侧的解析逻辑测试与编译检查在 `src-tauri` 目录下运行：

```bash
cd src-tauri
cargo test
cargo check
```

## 数据来源

当前版本主要通过公开行情接口获取市场数据，代码中已接入：

- 腾讯行情相关接口
- 新浪搜索建议接口

这些接口并非由本项目维护，返回结构、可用性与频率限制都可能发生变化。如果你打算长期维护或商用，建议尽早抽象数据层并切换到更稳定、合规的正式数据服务。

## 设计目标

这个项目不是一个完整交易终端，而是一个偏“桌面辅助信息层”的工具。当前实现重点放在：

- 快速打开
- 轻量驻留
- 少操作路径
- 适合多任务工作场景

如果你希望把它继续扩展成更完整的桌面行情终端，比较适合继续演进的方向包括：

- 更多市场与指数源
- 更完整的技术指标体系
- 历史持久化与本地缓存
- 多窗口 / 多布局模式
- 告警与通知能力

## 开发建议

如果你准备继续维护这个仓库，建议优先关注以下几类问题：

- 行情接口字段变更
- 多屏 / 多分辨率下的窗口定位行为
- Tauri 权限与窗口能力配置
- 公共样式类名与 UnoCSS 的命名冲突
- 公开仓库中的品牌、图标与截图素材整理
