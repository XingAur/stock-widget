# A+ Assistant

A+ Assistant 是一个面向 Windows 的股票和基金桌面行情助手，基于 Tauri 2、Vue 3、TypeScript 和 Rust 构建。当前最新正式版为 `1.0.2`，默认安装到 D 盘，适合常驻桌面查看自选资产、快速刷新行情、维护持仓盈亏和低打扰跟踪市场。

## 正式版信息

- 版本：`1.0.2`
- 平台：Windows 10 / Windows 11
- 安装包：NSIS x64
- 默认安装目录：`D:\A+助手`
- 升级安装：检测到旧版本时会中文提示升级，并默认保留自选、设置和持仓数据
- 最新安装包：`src-tauri/target/release/bundle/nsis/A+助手_1.0.2_x64-setup.exe`
- 应用名称：`A+助手`
- 标题切换：点击标题在 `A+Stock Assistant` 和 `A+Fund Assistant` 间切换
- 图标：安装包图标、程序图标、托盘图标统一使用默认图标资源

## 1.0.2 正式版更新

- 新增股票和基金双模式：点击标题在 `A+Stock Assistant` / `A+Fund Assistant` 间切换。
- 新增场外开放式基金自选，支持搜索、添加、删除、排序和估算净值刷新。
- 新增股票和基金持仓录入：股票按成本价和股数计算盈亏，基金按持有金额和收益计算盈亏。
- 优化列表排序：股票和基金均支持左侧手柄拖拽，保留右键菜单排序。
- 优化设置可靠性：置顶、自启、恢复默认增加 pending、失败回滚和启动重放。
- 修复安装后白屏风险：允许 Tauri IPC 使用 `ipc:` / `http://ipc.localhost`，避免 WebView2 CSP 拦截内部调用。
- 增加本地设置清洗：升级保留数据时会过滤异常主题、透明度、颜色和字体配置。
- 清理自启占位参数，避免开机自启时携带无意义命令行参数。
- 修复基金估算涨跌幅缺失时仍显示红绿方向的问题，缺失值现在保持中性。
- 修复全局成功/危险颜色变量反转，设置页关闭按钮等危险态恢复为红色。

## 主要功能

### 股票自选

- 支持股票名称和代码搜索。
- 支持添加、删除、右键排序、拖拽手柄排序。
- 支持 30 秒自动刷新。
- 股票详情页展示分时图、K 线、均线、成交指标和买卖五档。
- 右键股票可录入成本价和股数，列表悬停到该股票时展示市值、成本、盈亏和盈亏比例。

### 基金自选

- 仅支持场外开放式基金，不使用股票行情接口中的基金、ETF、LOF、场内基金或封闭式基金。
- 支持基金名称和代码搜索。
- 支持添加、删除、右键排序、拖拽手柄排序。
- 列表展示基金名称、代码、当日估算涨跌幅和估算净值。
- 右键基金可录入持有金额和收益，列表悬停到该基金时展示持有金额、成本、盈亏和盈亏比例。
- 基金第一版只做列表实时估算，不提供基金详情页、历史净值图和 K 线。

### 列表交互

- 股票和基金各自保存自选列表和排序。
- 拖拽只能从卡片左侧手柄触发，避免和点击详情、删除按钮冲突。
- 拖动启动阈值为 6px，并提供目标位置高亮。
- 股票点击打开详情；基金点击不打开详情，仅保留列表交互。
- 删除按钮只在鼠标悬停、焦点或选中状态出现，默认布局靠右显示行情数据。

### 设置与窗口

- 支持深色、浅色和自定义主题。
- 支持透明度、字体、指数 sparkline、最小化到托盘、窗口置顶和开机自启。
- 置顶和开机自启有 pending 状态，快速连点不会造成状态错乱。
- 系统调用失败时会回滚 UI 开关状态。
- 恢复默认会同步关闭置顶和自启；副作用失败时不会覆盖本地设置。
- 启动后会重新应用置顶状态，保证 UI 与窗口实际状态一致。
- 启动时会清洗旧版本或异常写入的本地设置，降低升级后透明窗口、异常主题和白屏风险。
- 浅色模式下股票详情页的图表、K 线、买卖五档和 tooltip 做了专门颜色适配。

## 数据源说明

当前版本使用公开接口获取行情和搜索数据：

- 股票行情：腾讯行情相关接口
- 股票搜索：新浪搜索建议接口
- 基金搜索：东方财富基金搜索接口
- 基金估算：天天基金 `fundgz.1234567.com.cn/js/{code}.js`

这些接口不由本项目维护，返回结构、可用性、频率限制和估算更新时间都可能变化。基金实时更新按开放式基金估算净值和估算涨跌幅理解，不等同于交易所逐笔价格。接口异常时会尽量保留旧数据，单条基金刷新失败不会阻塞其他基金。

## 开发环境

- Node.js 18+
- Rust stable
- Windows 10 / Windows 11

安装依赖：

```bash
npm install
```

前端开发：

```bash
npm run dev
```

桌面应用开发：

```bash
npm run tauri:dev
```

正式构建：

```bash
npm run tauri:build
```

构建完成后，安装包位于：

```text
src-tauri/target/release/bundle/nsis/A+助手_1.0.2_x64-setup.exe
```

## 维护说明

- 本项目只提供行情展示和自选维护，不提供交易能力。
- 股票和基金数据仅用于信息展示和技术研究，不构成投资建议。
- 若需要长期商用，建议抽象数据层并切换到稳定、合规的正式数据服务。
- Windows 安装版依赖系统 WebView2 运行时；如极端情况下安装后仍出现白屏，可先关闭程序后重新启动，或清理 `%LOCALAPPDATA%\com.stock.widget\EBWebView` 后再试。
